# Express + Typescript Server

Servidor de express que usa typescript para desarrollar un api con Mongo DB

## Comandos

- Bajar dependencias

```bash
yarn
```

- Correr Desarrollo

```bash
yarn dev
```

- Docker Desarrollo

```bash
yarn db:up
yarn db:down
yarn db:destroy
```

- Correr Pruebas

```bash
yarn test
yarn coverage
```

- Buildear Produccion

```bash
yarn build
yarn serve
```
